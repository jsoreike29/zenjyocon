## 3. キーボード配列デザイナー (`keyboard_designer.html`) の使い方

最適化ツールで生成された配列や、既存の配列を手動でカスタマイズするためのビジュアルエディタです。「数値的には最適だが、感覚的に押しづらいキー」を直感的に入れ替えることができます。

### 手順

1.  **ファイルの起動**: `keyboard_designer.html` をブラウザで開きます。
2.  **配列の読み込み** (任意):
    * `JSON読み込み` ボタンを押し、最適化ツールで作成した `layout.json` を選択すると、その配列状態から編集をスタートできます。
    * 何も読み込まない場合は、標準のQWERTY配列からスタートします。
3.  **キーの入れ替え**:
    * 移動させたいキーをクリックして選択状態にします（色が変化します）。
    * 次に、移動先のキーをクリックします。
    * 2つのキーの内容が入れ替わります。これを繰り返して理想の配置を作ります。
4.  **結果の保存**:
    * 編集が終わったら `JSON保存` ボタンを押します。
    * 編集後の配列データ（例: `my_custom_final.json`）がダウンロード、またはコードが表示されます。
    * このファイルは、**キーボード比較ヒートマップ**で読み込んで再評価することができます。

## 🔄 推奨ワークフロー

3つのツールを組み合わせて、最強の配列を作成する流れは以下の通りです。

1.  **生成 (`optimizer`)**: `keyboard_optimizer_json.html` で、自分の文章データに基づいた「理論上の最強配列」を計算し、JSONを保存。
2.  **調整 (`designer`)**: `keyboard_designer.html` にJSONを読み込み、「小指がきつい」「ショートカットが押しにくい」などの箇所を手動で微調整して保存。
3.  **評価 (`heatmap`)**: `keyboard_heatmap.html` で調整後のJSONを読み込み、ヒートマップで無理な指の動きがないか最終確認する。
