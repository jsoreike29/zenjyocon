## 3. キーボード配列デザイナー (`keyboard_designer.html`) の使い方

最適化ツールで生成された配列や、既存の配列を手動でカスタマイズするためのビジュアルエディタです。
**最大の特長は、AutoHotKey (AHK) 形式での出力に対応している点です。これにより、作成した配列をWindows上で「今すぐ」使い始めることができます。**

### 手順

1.  **ファイルの起動**: `keyboard_designer.html` をブラウザで開きます。
2.  **配列の読み込み** (任意):
    * `JSON読み込み` ボタンを押し、最適化ツールで作成した `layout.json` を選択すると、その配列状態から編集をスタートできます。
3.  **キーの入れ替え**:
    * 移動させたいキーをクリックして選択し、移動先をクリックして入れ替えます。
4.  **結果の保存と適用**:
    * **すぐに使いたい場合 (`AHK保存`)**:
        * `AHK保存` ボタンを押して `.ahk` ファイルをダウンロードします。
        * **使い方**: [AutoHotKey](https://www.autohotkey.com/) がインストールされたPCで、ダウンロードしたファイルをダブルクリックするだけです。
        * レジストリ変更やPCの再起動は不要。スクリプトを停止すればすぐに元の配列に戻るため、気軽にテストできます。
    * **保存・再編集用 (`JSON保存`)**:
        * 後でまた編集したり、ヒートマップで解析したい場合は `JSON保存` も行っておいてください。

## 🔄 推奨ワークフロー

1.  **生成 (`optimizer`)**: 文章データから理論上の最強配列を計算し、JSON出力。
2.  **調整 & 試用 (`designer`)**: JSONを読み込んで微調整し、**AHK形式で出力して実際にタイプして試す。**
    * *「やっぱりこのキーはここが良い」と思ったら、スクリプトを止めてブラウザで修正し、再度AHK出力して試す...というサイクルを回せます。*
3.  **評価 (`heatmap`)**: 最終的に確定した配列のJSONを読み込み、指の負担を視覚的に確認する。
