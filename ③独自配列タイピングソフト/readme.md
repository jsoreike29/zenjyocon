## 3. 独自配列タイピング練習 (`keyboard_typing.html`) の使い方

作成した配列（JSON）を読み込んで、すぐにタイピング練習ができるツールです。
OSのキーボード設定を変更する必要がないため、「まだ移行するか決めていないが、試しに打って感触を確かめたい」という段階で非常に役立ちます。

### 手順

1.  **ファイルの起動**: `keyboard_typing.html` をブラウザで開きます。
2.  **配列の読み込み**:
    * 画面上の `配列読込(JSON)` ボタンを押し、最適化ツールやデザイナーで保存した `layout.json` を選択します。
    * ※読み込まない場合は、通常のQWERTY配列での練習になります。
3.  **練習用テキストの読み込み** (任意):
    * デフォルトでも英語の短文が収録されていますが、`テキスト読込(.txt)` ボタンから好きな英文テキストファイル（コードや小説など）を読み込んで練習に使えます。
4.  **練習開始**:
    * `スタート` ボタン（またはスペースキー）を押して練習を開始します。
    * 画面上のキーボードには**独自配列のキー配置**が表示されます。それを見ながら、指の動きを馴染ませてください。

### 特徴
* **可視化ガイド**: 画面上のキーボードが、次に押すべきキーをハイライトして教えてくれます。
