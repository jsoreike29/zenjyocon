## カスタムキーボード配列構築ツールセット

ブラウザ上で完結する、独自のキーボード配列を作成・調整・評価・練習するためのHTMLツール群です。
個人の文章入力の癖に合わせて、指の移動距離や疲労を最小化する「最強の配列」を構築するために設計されています。

## 📂 収録ファイルと機能概要

### 1. 配列最適化ツール (Optimizer)
* **ファイル名**: `keyboard_optimizer_json.html`
* **概要**: 
  入力されたテキストデータを解析し、焼きなまし法（Simulated Annealing）アルゴリズムを用いて、理論上最も効率的なキー配置を自動生成します。
* **主な機能**:
  * 指の移動距離、同指連打、同手連続打鍵に対するペナルティ計算。
  * 最適化された配列データのJSON出力。

### 2. 配列デザイナー (Designer)
* **ファイル名**: `keyboard_designer.html`
* **概要**: 
  キーボード配列を視覚的に編集するためのGUIエディタです。最適化ツールで生成された配列を読み込み、人間の感覚に合わせて微調整を行うために使用します。
* **主な機能**:
  * ドラッグ＆ドロップ感覚でのキー入れ替え。
  * **AutoHotKey (.ahk) 形式での出力**（Windows環境ですぐに配列を利用可能にする機能）。
  * 編集データのJSON保存・読み込み。

### 3. ヒートマップ可視化 (Heatmap)
* **ファイル名**: `keyboard_heatmap.html`
* **概要**: 
  特定のテキストを入力した際のキー使用頻度をヒートマップとして可視化し、配列の効率性を評価します。
* **主な機能**:
  * 一般的なQWERTY配列と、作成した独自配列（JSON）の比較。
  * ホームポジション使用率や指ごとの負担分布の確認。

### 4. 独自配列タイピング練習 (Typing)
* **ファイル名**: `keyboard_typing.html`
* **概要**: 
  OSの設定を変更することなく、ブラウザ上で独自配列のタイピング練習ができるアプリケーションです。
* **主な機能**:
  * 独自配列データ（JSON）の読み込みと適用。
  * 任意のテキストファイルを使用した練習。
  * 次に打つキーのハイライト表示（可視化ガイド）。

---

## 🔄 ワークフローの全体像

本ツールセットは、以下のサイクルで配列を完成させることを想定しています。

1.  **生成**: `Optimizer` で理論値を計算し、ベースとなる配列を作成。
2.  **調整**: `Designer` で使いにくい箇所を手動修正し、AHKファイルを出力して実用性をテスト。
3.  **評価**: `Heatmap` で修正が指の負担に悪影響を与えていないか客観的に確認。
4.  **習得**: `Typing` で指に配列を覚え込ませる。

## ⚙️ 技術仕様・特徴

* **ローカル処理**: 入力したテキストデータなどがサーバーに送信されることはありません。
* **相互運用性**: 各ツール間はJSON形式のデータで連携可能です。
